# MNIST Digit Classification

This project is based on Artificial Intelligence and Machine Learning and focuses on recognizing handwritten digits from 0 to 9.

## Tools Used
- Python
- TensorFlow
- NumPy
- Matplotlib

## Description
The model is trained on the MNIST dataset and achieves around 95–97% accuracy.

## How to Run
Open the notebook in Google Colab and run all cells.
